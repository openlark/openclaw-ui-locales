## 概览

**OpenClaw UI Locales**是一款为OpenClaw Web 控制界面提供中文、日文和韩文等语言转换，让您彻底告别语言障碍，以最熟悉的语言高效操控设备。


## 安装

```bash
# 克隆仓库
git clone https://github.com/openlark/openclaw-ui-locales.git
cd openclaw-ui-locales

# 安装依赖包
npm install

# 执行命令
npm run locale
```
