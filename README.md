## Overview

**OpenClaw UI Locales** is a multilingual switching tool specifically designed for the OpenClaw Web control interface. It allows you to seamlessly switch between multiple languages with a single click, completely eliminating language barriers and enabling you to efficiently control your device in the language you are most familiar with.

## Install

```bash
# Clone the repository
git clone https://github.com/openlark/openclaw-ui-locales.git
cd openclaw-ui-locales
# Install dependencies
npm install
# Execute the command
npm run locale
```
