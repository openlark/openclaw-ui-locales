## 概覽

**OpenClaw UI Locales** 是專為 OpenClaw Web 控制介面打造的多語言切換工具，讓您一鍵在 10+ 國家語言間無縫切換，徹底告別語言障礙，始終用最熟悉的語言高效掌控設備。

支援的國家語言如下：

```
簡體中文        - zh-CN
繁體中文        - zh-TW
日語            - ja
韓語            - ko
法語            - fr
德語            - de
義大利語        - it
葡萄牙語        - pt
西班牙語        - es
越南語          - vi
菲律賓語        - fi
泰盧固語        - te
印地語          - hi
```

## 安裝

```bash
# 克隆倉庫
git clone https://github.com/openlark/openclaw-ui-locales.git
cd openclaw-ui-locales

# 安裝依賴套件
npm install

# 執行命令
npm run locale
```
