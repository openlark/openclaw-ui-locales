## 概览

**OpenClaw UI Locales**是专为OpenClaw Web控制界面打造的多语言切换工具，让您一键在中文、日语、韩语、法语、孟加拉语等语言间无缝切换，彻底告别语言障碍，始终用最熟悉的语言高效掌控设备。


## 安装

```bash
# 克隆仓库
git clone https://github.com/openlark/openclaw-ui-locales.git
cd openclaw-ui-locales

# 安装依赖包
npm install

# 执行命令
npm run locale
```
